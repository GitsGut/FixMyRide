FixMyRide is a full-stack vehicle service management platform that allows 
users to book car and bike maintenance services online. 

Features:
- User registration with OTP verification (Twilio SMS)
- JWT-based authentication and role-based access (User, Admin, Super Admin)
- Browse and book vehicle services (Car & Bike)
- Admin panel to manage services, users and appointments
- Razorpay payment integration
- Vehicle management system
- Feedback system

Tech Stack:
- Backend: Java Spring Boot, Spring Security, Hibernate JPA, MySQL
- Frontend: Angular, TypeScript, Tailwind CSS
- SMS: Twilio
- Payment: Razorpay
- Authentication: JWT
